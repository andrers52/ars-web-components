HTML Usage:

Event Handling:

Event: ars-button:${id}:click
Detail: { result: "clicked" }

Attributes:

effect-color: Color for the pressed animation effect (optional).
CSS Classes: Use your own CSS classes for styling - component is styling-agnostic.
