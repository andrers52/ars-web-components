About Remote Call Mixin

This mixin enables components to call methods on other components remotely using custom events. It consists of two parts: RemoteCallCaller for sending method calls and RemoteCallReceiver for receiving and executing them. Perfect for decoupled component communication using component IDs.

üìñ Implementation Guide

How to Use the Remote Call Mixin:

1. Import the mixins: import RemoteCallCaller from './remote-call-caller.js' and import RemoteCallReceiver from './remote-call-receiver.js'

2. Extend your caller class: class MyCaller extends RemoteCallCaller(HTMLElement)

3. Extend your receiver class: class MyReceiver extends RemoteCallReceiver(HTMLElement)

4. Give your receiver component an ID: <my-receiver id="my-receiver"></my-receiver>

5. Call remote methods: this._callRemote('my-receiver', 'methodName', arg1, arg2)

6. Remember to call super.connectedCallback() and super.disconnectedCallback() in receiver components

// Caller Component import RemoteCallCaller from './remote-call-caller.js'; class MyCaller extends RemoteCallCaller(HTMLElement) { constructor() { super(); this.innerHTML = '<button onclick="this._callRemote(\'my-receiver\', \'increment\')">Increment</button>'; } } customElements.define('my-caller', MyCaller);
// Receiver Component import RemoteCallReceiver from './remote-call-receiver.js'; class MyReceiver extends RemoteCallReceiver(HTMLElement) { constructor() { super(); this.counter = 0; this.innerHTML = `<div>Counter: <span id="counter">${this.counter}</span></div>`; } connectedCallback() { super.connectedCallback(); // Important! } increment() { this.counter++; this.querySelector('#counter').textContent = this.counter; } decrement() { this.counter--; this.querySelector('#counter').textContent = this.counter; } } customElements.define('my-receiver', MyReceiver); // Usage in HTML: // <my-receiver id="my-receiver"></my-receiver>

‚ú® Features

What the Remote Call Mixin Provides:

  • Decoupled Communication: Components can communicate without direct references
  • Method Calling: Call any public method on any component by ID
  • Argument Passing: Pass multiple arguments to remote methods
  • Security: Private methods (starting with _) are automatically blocked
  • Error Handling: Graceful error handling for non-existent methods
  • Event-Based: Uses custom events for reliable communication
  • Simple Setup: Just extend the mixin and give your receiver an ID
  • Lifecycle Management: Proper cleanup in disconnectedCallback
