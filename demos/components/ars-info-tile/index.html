ars-info-tile

Structured information display tile for dashboards and graph visualizations.

About

Renders a card with title, subtitle, accent color header, and key-value property list. Supports selection and dragging states for graph visualization integration.

Usage

const tile = document.querySelector('ars-info-tile'); tile.data = { id: 'bot_001', title: 'Scalper Bot', subtitle: 'Trading Bot', accentColor: '#3aa8ff', properties: { status: 'active', pnl: '+$1,234', trades: '48' } };
