ars-bottom-nav

Mobile-optimized bottom tab navigation bar.

← Back to Demo Page

About

Parent <ars-bottom-nav> manages active state across <ars-bottom-nav-item> children. Each item supports icon and label slots. Ideal for mobile-first navigation patterns.
